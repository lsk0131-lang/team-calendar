3 각 섹션 기능 설명

🔢 상단 KPI 카드

대시보드 최상단에 7개의 핵심 지표가 표시됩니다. 커버리지 카드(1·2·3티어)에 마우스를 올리면 충분/보통/부족 기준이 툴팁으로 나타납니다.

ℹ️
인당 건수 계산 기준

해당 기간 내 실제 미팅이 있는 팀원 수로 나눈 평균값입니다.

📅 연도 / 월 필터

상단의 2026년 / 2025년 / 2024년 버튼으로 연도를 선택하고, 📅 월별 드롭다운으로 전체 또는 특정 월을 선택합니다. 전체를 선택하면 해당 연도 1월부터 현재까지의 누적 데이터를 표시합니다.

🔄 매체 싱크

구글 스프레드시트의 매체 구분표와 대시보드 매체 목록을 동기화합니다. 매체 티어 변경 또는 신규 매체 추가 후 클릭하면 즉시 반영됩니다. 싱크 완료 후 데이터를 자동으로 새로고침합니다.

🔥 매체별 미팅 현황 (히트맵)

각 매체의 월별 미팅 횟수를 색상 강도로 나타냅니다. 진할수록 횟수가 많습니다. 티어별 필터(전체 / 1티어 / 2티어 / 3티어 / 기타)로 원하는 범위만 볼 수 있으며, 매체명 클릭 시 해당 매체의 상세 미팅 내역 모달이 열립니다.

📰 매체별 미팅

미팅 횟수 기준 Top 20 매체가 표시됩니다. 두 가지 필터를 조합해 사용할 수 있습니다.

  • 시간대 탭 (전체 / 🍱 점심 / 🍷 저녁) — 점심(오전 11시~오후 2시) 또는 저녁(오후 5시 이후) 미팅만 필터링
  • 티어 탭 (전체 / 1티어 / 2티어 / 3티어 / 기타) — 해당 티어 매체만 표시. 두 필터를 조합하면 예: 1티어 점심 미팅만 확인 가능

바에 마우스를 올리면 날짜 / 팀원 / 기자명 상세 목록이 표시됩니다. 점심·저녁 탭 선택 시 해당 시간대 미팅만 툴팁에 표시됩니다.

✏️ 기자별 미팅 Top 20

가장 많이 만난 기자 20명을 순위별로 표시합니다. 바에 마우스를 올리면 누가 언제 만났는지 확인할 수 있습니다.

🎯 관리 매체 커버리지 현황

1·2·3티어 관리 매체의 미팅 충족 현황을 색상 점으로 표시합니다. 각 티어 컬럼은 내용이 넘칠 경우 스크롤로 확인할 수 있습니다.

색상의미1티어 기준2티어 기준3티어 기준
🟢 충분목표 이상월 4회 이상월 3회 이상월 2회 이상
🟠 보통준수월 2~3회월 2회월 1회
🔵 부족미달월 1회 이하월 1회 이하월 0회
⚠️
전체 기간 조회 시 경과 월수에 비례 적용

예: 5월까지 조회 시 1티어 충분 기준 = 4회 × 5개월 = 20회

🏆 분기별 랭킹

분기별(Q1~Q4) + 연간 누적으로 아래 5개 항목의 Top 3를 표시합니다.

  • 📊 최다 미팅 크루 — 전체 매체 미팅 횟수 기준
  • 🍱 점심 미팅 크루 — 오전 11시~오후 2시 사이 미팅 횟수 기준 (매체 미팅 포함 전체)
  • 🍷 간 건강 염려 크루 — 오후 5시 이후 저녁 미팅 횟수 기준
  • 📰 개고생 크루 — 기타 매체(비관리 매체) 미팅 횟수 기준. 마우스 오버 시 만난 매체 목록 확인 가능
  • 📅 최다 회의 크루 — 오전 9시~오후 6시 사이 비매체 내부 미팅 횟수 기준 (휴가·연차 제외)

⬇️ 엑셀 다운로드

우측 상단 엑셀 다운로드 버튼을 클릭하면 현재 선택된 연도·월의 데이터를 엑셀 파일로 받을 수 있습니다. 3개 시트로 구성됩니다.

  • 전체 로데이터 — 이벤트별 전체 원본 데이터
  • 매체별 집계 — 매체별 월간 미팅 횟수
  • 팀원별 집계 — 팀원별 월간 미팅 횟수
4 캘린더 입력 규칙 ★

대시보드 데이터는 구글 캘린더 이벤트 제목을 자동으로 파싱합니다.
아래 규칙을 지켜야 정확하게 집계됩니다.

✅
기본 형식: 이니셜 + 매체명 + 기자명

예) La 조선일보 김강한  /  La 조선 김강한 (약자 OK)

입력 예시 판정 설명
La 조선일보 김강한 ✓ 이니셜 + 매체 정식명 + 기자명
La 조선 김강한 ✓ 매체 약자도 인식
La 조선 김강한, 최아리 ✓ 복수 기자는 쉼표로 구분
LaH 조선 김강한 ✓ 복수 참여 — 붙여쓰기 가능 (공백 구분도 OK)
La H 조선 김강한 ✓ 복수 참여 — 공백으로 구분
La 조선 팀미팅 ✗ 팀미팅 대신 기자명 모두 기입
La 조선 데스크+김강한 ✗ + 대신 쉼표로 구분
La 조선 산업부(김강한, 최아리) ✗ 괄호·부서명 제외, 기자명만 기입
La 조선 김강한 기자 ✗ 직함('기자') 제외 — 이름만 입력

5 커버리지 기준

관리 매체 커버리지는 월 미팅 횟수를 기준으로 충분 / 보통 / 부족 / 없음으로 분류합니다.
전체 기간 조회 시 경과 월수에 비례해 기준이 높아집니다.

티어 충분 🟢 보통 🟠 부족 🔵
1티어 조선·중앙·동아·매경 등 (9개) 월 4회 이상 월 2~3회 월 1회 이하
2티어 한겨레·경향·서울경제 등 (24개) 월 3회 이상 월 2회 월 1회 이하
3티어 전자신문·지디넷 등 (47개) 월 2회 이상 월 1회 월 0회
⚠️
전체 기간 조회 시 경과 월수에 비례 적용

예: 5월까지 조회 시 1티어 충분 기준 = 4회 × 5개월 = 20회

커버리지 비율(%) = 충분 + 보통 매체 수 ÷ 전체 관리 매체 수 × 100

6 자주 묻는 질문

Q. 캘린더에 입력했는데 대시보드에 반영이 안 돼요

대시보드는 실시간이 아닌 새로고침 시점에 데이터를 가져옵니다. 상단의 ↻ 새로고침 버튼을 눌러보세요. 그래도 안 보인다면 이니셜·매체명 형식이 규칙에 맞는지 확인해주세요.

Q. 같이 간 미팅은 각자 캘린더에 입력해야 하나요?

아닙니다. 한 명만 입력하되 이니셜을 모두 적으면 됩니다.
예) LaH 조선 김강한 → 레일라와 휴 둘 다 집계됩니다.

Q. 미팅 건수가 실제와 달라요

같은 날 같은 제목의 이벤트라도 팀원별로 각각 1건씩 집계됩니다. 단, 매체 미팅 횟수 자체는 동일 이벤트를 1건으로 처리합니다.

Q. 기타 매체란 무엇인가요?

1·2·3티어 관리 매체 외에 인식된 매체입니다. 매체별 미팅 차트 기타 탭에서 확인할 수 있으며, 커버리지 분석에는 포함되지 않습니다. 개고생 크루 랭킹의 집계 대상이기도 합니다.

Q. 매체 목록을 변경했는데 반영이 안 돼요

구글 스프레드시트에서 매체 구분을 수정한 후 대시보드 상단의 🔄 매체 싱크 버튼을 클릭하면 즉시 반영됩니다. 싱크 후 데이터가 자동으로 새로고침됩니다.

Q. 광고·행사 미팅은 어떻게 처리되나요?

[광고]·[행사] 등 대괄호로 표기된 이벤트와, 기자명 자리에 광고·행사·포럼·조찬·심포지엄 등 키워드가 있는 미팅은 집계에서 자동 제외됩니다.

Q. 테마를 바꿀 수 있나요?

상단 헤더의 🌙 버튼을 클릭하면 다크 ↔ 라이트 모드로 전환됩니다. 선택한 테마는 브라우저에 저장됩니다.

Q. 문의 / 오류 신고

데이터 오류나 기능 관련 문의는 레일라에게 연락해주세요.
